# Ouroboros

Self-Improving AI Workflow System

## Overview

Ouroboros is a self-improving AI workflow system designed to enhance and automate development processes through intelligent iteration and feedback loops.

## Requirements

- Python >= 3.14

## Installation

```bash
uv sync
```

## Usage

```bash
uv run ouroboros
```

## Development

```bash
# Install development dependencies
uv sync --all-groups

# Run tests
uv run pytest

# Run type checking
uv run mypy src/

# Run linting
uv run ruff check src/
```
