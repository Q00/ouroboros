# 30-Second Quick Start Demo Script

## Overview
**Duration**: 30 seconds
**Goal**: Show the fastest path from zero to running Ouroboros
**Target Audience**: Developers evaluating Ouroboros

## Script Breakdown

| Time | Scene | Voiceover / Action |
|------|-------|-------------------|
| 0-5s | Terminal with plugin installed | "Ouroboros is a Claude Code plugin that gives you autonomous workflow execution. Here's how to get started in 30 seconds." |
| 5-15s | Type `ooo run` and watch execution | "Simply type `ooo run` and describe what you want. Ouroboros plans, executes, and verifies your task autonomously." |
| 15-25s | Show completion with results | "Watch as it breaks down the work, runs agents in parallel, and delivers verified results with a summary." |
| 25-30s | Show final state / CTA | "Ready to automate your development workflow? Type `ooo welcome` to begin." |

## Terminal Commands for Demo

```bash
# Pre-setup (do this before recording)
# Ensure Ouroboros is installed and configured
claude /plugin install ouroboros@ouroboros

# Demo command (this is what you type on camera)
ooo run Create a simple counter component with increment/decrement buttons
```

## Visual Setup

### Terminal Configuration
```bash
# Recommended terminal settings
- Font: SF Mono, MesloLGS NF, or JetBrains Mono (14-16pt)
- Theme: Dark background with high contrast
- Window size: 1280x720 minimum (for 720p video)
- Hide: Toolbar, scrollbar, menubar
```

### Pre-Demo Checklist
- [ ] Plugin installed and verified

- [ ] Recording software configured and tested

## Post-Production Notes

### Editing Tips
1. **Trim excess**: Remove pauses and thinking time
2. **Speed up**: Consider 1.2x speed for typing sections
3. **Highlight**: Use zoom/callouts for key moments
4. **Music**: Add subtle background music for intro/outro

- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Format**: MP4 (H.264)
- **Frame Rate**: 30fps
