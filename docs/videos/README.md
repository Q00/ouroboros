# Ouroboros Video Demo Guide

This directory contains scripts and guides for creating video demos of Ouroboros.

## Contents

- `script.md` - 30-second Quick Start demo script
- `screen-recording-guide.md` - Step-by-step screen recording guide
- `asciinema-guide.md` - Terminal recording setup with asciinema
- `checklist.md` - Preparation checklist for demo creation

## Quick Reference

### Demo Types

1. **Quick Start (30s)** - Fastest path to value
2. **Feature Deep Dive (2-3min)** - Detailed feature walkthrough
3. **End-to-End Workflow (5min)** - Complete use case demonstration

### Recording Tools

- **macOS**: Built-in screen recorder (Cmd+Shift+5)
- **Linux**: SimpleScreenRecorder, OBS Studio
- **Terminal**: asciinema (recommended for CLI demos)
- **Cross-platform**: OBS Studio

## File Naming Convention

```
{demo-type}_{version}_{date}.{ext}
Example: quickstart_v1_20250111.mp4
```
